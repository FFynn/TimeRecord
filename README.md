# TimeRecord

◆ 这是一个支持文本录入编辑的工具、主要用于图文制作以及制作完成后的导出与分享;

◆ 项目正在持续完善...

◆ 此项目文本样式以及图片素材来源于iOS版「萝卜书摘」，仅做个人练习使用，如有侵犯请联系本人删除:-)

---
